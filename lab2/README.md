# Лабораторная работа: потоковое шифрование на LFSR

Веб-приложение для шифрования и дешифрования файлов любого формата с помощью ключевого потока на основе LFSR.

## Что реализовано

- Вариант 15: регистр длиной `37` бит.
- Многочлен по умолчанию: `x^37 + x^12 + x^10 + x^2 + 1`.
- Начальное состояние регистра по умолчанию: все единицы.
- Поле seed принимает только `0` и `1`, остальные символы автоматически удаляются.
- Длина seed ограничена 37 символами; если меньше 37, строка дополняется нулями.
- Проверка на запрет ключа из одних нулей.
- Шифрование/дешифрование файлов любого расширения (побайтный XOR с ключевым потоком).
- Вывод в интерфейсе:
  - ключ (симметрично: первые и последние N бит),
  - исходный файл в двоичном виде (превью),
  - результат в двоичном виде (превью).
- Сохранение результата на диск как байтового потока.

## Структура проекта

- `index.html` — разметка интерфейса.
- `css/main.css` — стили.
- `js/constants.js` — константы варианта.
- `js/lfsr.js` — логика LFSR.
- `js/binary.js` — утилиты преобразования в биты.
- `js/crypto.js` — XOR-обработка.
- `js/ui.js` — работа с элементами интерфейса.
- `js/app.js` — инициализация и обработчики.
- `Тесты_ЛР2_LFSR.xlsx` — тест-кейсы и лист проверки шагов LFSR.

## Как запустить

### Вариант 1: просто открыть файл

1. Открой `index.html` в браузере двойным кликом.

### Вариант 2: через локальный сервер (рекомендуется)

1. В корне проекта запусти:
   - `python3 -m http.server 8000`
2. Открой в браузере:
   - `http://localhost:8000`

## Как пользоваться

1. Укажи начальное состояние регистра (или нажми кнопку "Поставить все 1").
2. При необходимости измени многочлен (поле со степенями).
3. Выбери файл.
4. Нажми:
   - `Зашифровать` — для получения шифротекста,
   - `Расшифровать` — для обратного преобразования.
5. Скачай результат по ссылке "Скачать результат".

## Проверка корректности

1. Возьми любой файл (например, `txt`, `png`, `pdf`).
2. Зашифруй его.
3. Зашифрованный файл подай на вход "Расшифровать" с тем же seed и многочленом.
4. Итоговый файл должен совпасть с исходным побайтно.
