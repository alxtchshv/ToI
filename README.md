# 🔐 Криптографическая система

**Шифр Плейфейра (английский) + Шифр Виженера с прогрессивным ключом (русский)**

---

### 🚀 Как быстро запустить (самое главное)

1. Скачайте **все файлы** репозитория (или скачайте ZIP-архив).
2. Откройте файл **`index.html`** в любом браузере (Chrome, Edge, Firefox, Safari).

Готово! Программа работает полностью оффлайн, без установки и сервера.

---

## 📋 О проекте

Веб-приложение для лабораторной работы №1 по дисциплине **«Теория информации»** (вариант 9, БГУИР).

Реализованы два алгоритма шифрования:

- **Шифр Плейфейра** — текст на английском языке (5×5 матрица, I/J объединены)
- **Шифр Виженера с прогрессивным ключом** — текст на русском языке (алфавит 33 буквы, ключ сдвигается после каждого полного прохода)

### Возможности
- Шифрование и дешифрование
- Загрузка текста из `.txt` файла
- Встроенные тестовые примеры (включая все краевые случаи из отчёта)
- Сохранение результата в файл
- Копирование в буфер обмена
- Адаптивный красивый дизайн
- Полностью клиентская реализация (HTML + CSS + JavaScript)
